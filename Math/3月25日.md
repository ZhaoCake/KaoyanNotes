## 潜在问题分析（AI）

### 一、核心思维问题（根因）

1. 定义到结论的“最后一步”经常断裂
你已经能推到关键形式（如极限表达式、同阶无穷小表达式），但没有把它和定义/结论完成绑定，导致“会推不会判”。

2. 过早使用技巧，晚于使用主线
本该先走“定义-性质-定理”的稳定主线（单调有界、导数定义、拉格朗日后移项），却提前进入泰勒、变形技巧，增加了符号和计算风险。

3. 步骤跳跃，默认中间结论“显然成立”
典型表现是直接报答案符号或系数（如正负号、1/2 与 -1/2），省略了中间替换与方向判断，导致结果方向性错误。

4. 复合结构敏感度不足
遇到复合函数（如 1-t、3-x）时，链式法则和符号翻转没有自动触发检查，负号错误重复出现。

5. 收尾阶段注意力塌陷
最后一步移项、等号、系数最容易出错，说明你不是“不会”，而是“执行控制”在尾段掉线。

### 二、题型级预警（出现即报警）

1. 出现 x_{n+1}-x_n、递推、极限时
预警语：先定主线，不要先上泰勒。
动作：先判断“单调 + 有界”或“压缩映射/差值收敛”，再决定是否需要技巧。

2. 出现 f(a+h)、f(3-x)、f(1-t) 等替换时
预警语：先看自变量方向，先判符号再化简。
动作：单独写一行变量替换关系，如 h=3-x、dx=-dt，禁止心算跳步。

3. 用到洛必达、拉格朗日、等价无穷小时
预警语：定理用了，条件和收尾必须补齐。
动作：写出“条件是否满足 + 代回目标式 + 结论单位/符号”。

4. 进入最后 3 行计算时
预警语：最后三行 = 高风险区。
动作：强制慢写，逐项核对等号两边是否同步变化。

### 三、临场纠偏流程（60 秒）

1. 先说题目要我证明什么（收敛/单调/可导/极限值）。
2. 写主线模板（定义或基本定理）而不是技巧。
3. 每次变量替换后，先圈符号方向（尤其负号）。
4. 得到结果前，反代一次到原问题语句中。
5. 最后检查三件事：符号、系数、条件是否用全。

### 四、你的近期重点训练建议

1. 定义翻译训练
把“推出了某式”改成“因此按定义可得结论”，每天 3 题，强制写出“由定义知”。

2. 负号专项训练
专练含 3-x、1-t、复合函数求导与洛必达题，每题只盯“方向和负号”。

3. 收尾抗错训练
每题最后两步单独重写一遍，不看前文，只校验移项和系数。

### 五、量化预警指标（连续 7 天记录）

1. 跳步率：有多少题出现“中间结论未写”就直接给答案。
2. 符号错率：负号/方向错误占总错误比例。
3. 收尾错率：最后三行产生的错误占比。
4. 主线偏离率：本可用定义主线却先用技巧的题目比例。

目标：把“符号错率 + 收尾错率”压到总错题的 30% 以下；若连续 3 天超标，次日只做“定义 + 负号 + 收尾”专项，不开新题型。
